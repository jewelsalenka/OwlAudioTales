# 🦉 OwlAudioTales

Це мій застосунок для запису та прослуховування аудіоказок. Ідея зʼявилася з особистої потреби — хотілось мати простий і зручний інструмент, щоб створювати казки для своїх дітей прямо на телефоні.
Проєкт ще в процесі доробки, але він вже повністю функціональний.

## Як запустити

1. Клонувати репозиторій:
   ```bash
   git clone https://github.com/jewelsalenka/OwlAudioTales.git

2. Відкрити в Android Studio

3. Переконатися, що використовується:
	- Kotlin 2.x
	- Compose Compiler з відповідною версією
	- compileSdk 34
	- minSdk 26+

4. Запустити на емуляторі або фізичному девайсі (з доступом до мікрофона і камери)

---

## Що вміє застосунок

- Записувати аудіо (є можливість поставити на паузу і відновити)
- Вводити назву казки та імʼя автора
- Додавати обкладинку до запису (зробити фото або вибрати з галереї)
- Зберігати записані файли на пристрої
- Переглядати всі записи у бібліотеці
- Програвати казки, перемикатися на наступну або попередню
- Видаляти або редагувати записи (метадані й обкладинку)

---

## Технології

- Jetpack Compose
- MVVM
- Room для бази даних
- DataStore для налаштувань
- Navigation Compose
- MediaRecorder для запису аудіо
- Permissions API
